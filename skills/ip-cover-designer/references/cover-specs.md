# 封面尺寸参考

用户给出明确尺寸时始终优先服从用户。

| 场景 | 推荐尺寸 | 说明 |
| --- | --- | --- |
| GitHub README Hero | 1600 × 900 | 16:9，标题和角色放在中部安全区 |
| 网站 Hero | 1920 × 1080 | 预留响应式裁切空间 |
| 文章头图 | 1600 × 840 | 约 1.91:1，适合多数内容平台 |
| 社交横幅 | 1500 × 500 | 3:1，避免把脸和标题放在边缘 |
| 视频封面 | 1280 × 720 | 16:9，小尺寸下标题仍需清晰 |
| 方形封面 | 1080 × 1080 | 1:1，减少横向模块数量 |

## 安全区

- 标题与面部距离四边至少保留画面宽高的 5%。
- 横幅类封面将核心内容控制在中央 70%。
- 不把细小功能标签作为主要信息。
- 同一系列封面固定标题区、品牌署名和 IP 所在侧。
