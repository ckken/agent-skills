# 宣传图规格参考

用户或平台提供的实时规格优先于本表。

| 场景 | 推荐尺寸 | 用途 |
| --- | --- | --- |
| 横版发布图 | 1600 × 900 | GitHub、博客、发布页 |
| 公众号头图 | 900 × 383 | 宽幅文章头图 |
| 视频封面 | 1280 × 720 | 视频号、YouTube 等横版入口 |
| 方形宣传图 | 1080 × 1080 | 社交动态和产品卡片 |
| 竖版宣传图 | 1080 × 1440 | 移动端信息流 |
| Story 海报 | 1080 × 1920 | 全屏竖版传播 |

## 系列物料规则

- 先完成横版或方形母版，再派生其他尺寸。
- 主标题尽量不超过两行。
- IP 面部、产品主体和标题不得同时贴近裁切边缘。
- 竖版优先上下布局，横版优先左右布局。
- 小尺寸版本减少装饰和次要卖点，不缩成无法阅读的小字。
